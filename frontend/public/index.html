功能说明

1. 进制转换功能

  • 支持二进制、八进制、十进制、十六进制之间的相互转换
  • 提供栈实现、数组实现、递归实现三种算法
  • 实时可视化算法执行过程
  • 支持历史记录查看

2. 文件处理功能

  • 支持任意文件的二进制/十六进制查看
  • 提供文件校验功能（CRC32）
  • 支持拖拽上传文件

技术栈

  • 前端：原生 JavaScript + CSS3
  • 数据结构：栈、数组、递归
  • 文件处理：File API + Web Crypto API
  • 后端：Node.js + Express + MySQL
